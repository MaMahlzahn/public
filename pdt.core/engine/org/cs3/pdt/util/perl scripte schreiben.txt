Perl Scripte schreiben:

$_ stellt den aktuellen Buffer da:
	soll eine Zeile eines Files gelesen werden und der 
  	Inhalt der Zeile einer Variablen zugewiesen werden:
	while ( <CURRENTFILE> ) { # <CURRENTFILE> ist ein Handle, das mit open(CURRENTFILE, "$Filename") erzeugt wurde
		$text .= $_;  // .= concatiniert die aktuelle Zeile zu $text
	}
	
	Verwendet meine RegExpr, so wird diese auf den Inhalt von $_ angewandt:
	$str = s/[ab]*/$austaschstring/g
	

.= stringappend:
	$str .= "append"

special literals:
		 \w  Match a "word" character (alphanumeric plus "_")
         \W  Match a non-"word" character
         \s  Match a whitespace character
         \S  Match a non-whitespace character
         \d  Match a digit character
         \D  Match a non-digit character
         \pP Match P, named property.  Use \p{Prop} for longer names.
         \PP Match non-P
         \X  Match eXtended Unicode "combining character sequence",
             equivalent to C<(?:\PM\pM*)>
         \C  Match a single C char (octet) even under utf8.

Stelle ein \ für Sonderzeichen wie *,(,) u.ä., wenn sie als Zeichen in der Reg.Expr vorkommen sollen.
